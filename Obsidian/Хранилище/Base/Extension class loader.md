Body:

Что делает extension class loader?
!---
Ответ:
	- Загружает код в каталоги расширений (`<JAVA_HOME>/jre/lib/ext`, или любой другой каталог)
	- Управлять загрузкой расширений можно с помощью системной опции `java.ext.dirs`.
