#SQL 
Перечислите операторы определения данных (Data Definition Language (DDL))
!---
Ответ:
	- `CREATE` создает объект БД (базу, таблицу, представление, пользователя и т. д.)
	- `ALTER` изменяет объект
	- `DROP` удаляет объект
