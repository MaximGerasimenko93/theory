Существует ли возможность перенаправить потоки стандартного ввода/вывода?
!---
Ответ:
	- `setIn(InputStream)` - для ввода
	- `setOut(PrintStream)` - для вывода
	- `setErr(PrintStream)` - для вывода ошибок
